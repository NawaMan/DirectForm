Spec05B Users can access the fields.

  1. Users may want to access the field component.
  2. Accessing the elemtn by the property `fields` of the event object and property 'element'. For example, the event object is `event` and the input is `input1`, the user can access the input by:
    1. `event.fields.input1.element`
    2. `event.fields.['input1'].element`
