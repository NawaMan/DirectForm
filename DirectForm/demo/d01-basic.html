Basic

Description

  • This page aims to show the basic usage of DirectForm.
  • A direct form can be created from a container.
  • A variable is returned to represent the form. It can be used to submit the form.
  • When submitted, the value of any input components (except for button) in that container are automatically gathered and added to the event object in the property named "values" (e.g., `event.values['input1']` or `event.values.input1`).

Activity

  1. Enter value in the input field 'input1' and 'input2'.
  2. Click the 'Submit' botton.
  3. Observe the log message.
